== Description ==

This plugin created for you will be able to create an contact form with 
admin on your wordpress site.
[youtube https://www.youtube.com/watch?v=HdrYBTOC2Vs ]

== Installation ==

1. Upload `contact_form_123` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add link /?page=contact to menu on wordpress site
